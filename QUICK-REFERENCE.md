# QUICK REFERENCE - Daily Operations

## Match Day Checklist

### AT HALF-TIME (10 minutes total)
1. ⚡ Run your model → Get probability
2. 📝 Open predictions-data.json
3. ➕ Add new prediction at TOP:
```json
{
  "date": "2024-04-15",
  "match": "Team A vs Team B", 
  "league": "League Name",
  "prediction": "2H Over 1.5 Goals",
  "probability": 72,
  "odds": 1.85,
  "analysis": ["Point 1", "Point 2", "Point 3"],
  "result": "pending",
  "score": null
}
```
4. 💾 Save & upload to GitHub
5. 📱 Post to Twitter + Telegram (use template)

### AFTER FULL-TIME (2 minutes)
1. 📝 Find your prediction in predictions-data.json
2. ✏️ Update result:
   - Change "pending" → "win" or "loss"
   - Add score: "3-2 (2H: 2-1)"
3. 💾 Save & upload
4. 📱 Post result to Twitter + Telegram

## Emergency Troubleshooting

**Website not updating?**
- Check you uploaded to GitHub
- Wait 2-3 minutes for Pages to rebuild
- Clear your browser cache

**Stats showing wrong?**
- Check JSON format is correct
- No missing commas or brackets
- Date format: "YYYY-MM-DD"

**No followers?**
- Keep posting for 30+ days minimum
- Engage with other accounts
- Use hashtags (#Bundesliga #betting)
- Post during peak times
